My Neovim config with LSP, autocomplete and formatting configured.
1. Autocomplete : [blink.cmp](https://github.com/saghen/blink.cmp) 
2. LSP : [Mason](https://github.com/mason-org/mason.nvim) + [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig)
3. Formatting : Native `vim.lsp.buf.format()`

Kudos to [kickstart.nvim](https://github.com/nvim-lua/kickstart.nvim) for making the best documentation!
